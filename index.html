jQuery scrollsWith plugin

jQuery Sticky Sidebar Plugin

This is a sidebar!
by Scolling, it scolls with.
  • usage
  • options

Usage

1. insert sidebar and a invisible dummy-element
<div data-sidebar-dummy="sidebar1" ></div>
<div id="sidebar1" >...</div>

2. insert a invisible bottom-dummy-element on the place where scoll position ends
CONTENT...
<div data-sidebar-bottom="sidebar1" ></div>


3. add jQuery an the scrollsWith plugin
<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script&g
<script src="js/scrollsWith.js"></script>

4.init sidebar $("#sidebar1").scrollsWith(); //or
$("#sidebar1").scrollsWith({margin_top: 30, fixedClass: 'is-fixed'});

options

options can be set by the "option" function or by initialization.

dummy_attr (default: 'data-sidebar-dummy') The attrebut-name with identify the dummy element
bottom_dummy_attr (default: 'data-sidebar-bottom'). The attrebut-name with identify the dummy element with defined the end of the scolling area.
attr_value (default: null) the value of the dummy attrebutes. is null, then htey use the id of the element.
margin_top (default: 0) space top of the scoll-element.
fixedClass (default: 'fixed') Css class, thex will be added to the element if they static. The class is optional und will dont use by the scollWith Plugin to set the position of the element.
minWidth (default: 200) The minimal Browser with to enable fixing elements
